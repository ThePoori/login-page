# Login Page

A simple login page built with a Django backend, HTML, and CSS.

## 🚀 Features

- User login functionality powered by Django
- Clean and simple HTML/CSS frontend
- Lightweight project structure, easy to extend

## 🛠️ Tech Stack

- **Backend:** Python / Django
- **Frontend:** HTML, CSS

## 📦 Project Structure

```
login-page/
├── myproject/      # Django project files
├── .idea/          # IDE config (PyCharm)
└── README.md
```

## ⚙️ Setup & Run

1. Clone the repository
   ```bash
   git clone https://github.com/ThePoori/login-page.git
   cd login-page/myproject
   ```

2. Create a virtual environment and install dependencies
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install django
   ```

3. Run migrations
   ```bash
   python manage.py migrate
   ```

4. Start the development server
   ```bash
   python manage.py runserver
   ```

5. Open your browser at `http://127.0.0.1:8000/`
